## [Jacksum](http://jacksum.net/en/index.html)
	a free cross platform checksum utility, supports 58+ hash algorithms

下载来后安装，并配置好环境变量`/Applications/Jacksum`即可使用：  

```
# 计算文件的md5
jacksum -a md5 <filename>
```
**需要Java支持**
